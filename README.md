# 🎨 Paint Game Project

Modified version of the classic Paint game using Python Turtle and FreeGames.

## ✨ Features Added
- New colors
- Circle tool
- Rectangle tool
- Triangle tool

## 🛠 Technologies
- Python 3
- Turtle Graphics
- FreeGames
- Git & GitHub

## 📦 Installation

Install FreeGames:

```bash
pip install freegames
```

Or:

```bash
pip3 install freegames
```

## ▶ Run the Game

```bash
python3 activity_1.py
```

## 🎮 Controls

| Key | Action |
|------|--------|
| L | Line |
| S | Square |
| C | Circle |
| R | Rectangle |
| T | Triangle |
| U | Undo |
